# Приложение для полезных привычек 🌱
## Добро пожаловать в проект "Полезные привычки"! Это приложение, разработанное на Django Rest Framework (DRF), поможет вам следить за своими привычками и достигать поставленных целей. Пользуйтесь им на вебе или в Telegram! 🚀

## 📦 О проекте
Это приложение предназначено для того, чтобы помочь пользователям формировать и поддерживать полезные привычки. Благодаря простому и интуитивно понятному интерфейсу вы сможете:

Создавать и управлять привычками 📋
Отслеживать свой прогресс 📈
Получать напоминания о своих целях 🔔
Взаимодействовать с приложением через Telegram! 🤖

# 🎯 Установка
Следуйте этим шагам для установки проекта на вашем локальном компьютере:

## Клонируйте репозиторий:


1. git clone https://github.com/Jeirom/Habit.git
2. cd Habit


## Создайте виртуальное окружение и активируйте его:

python -m venv venv
source venv/bin/activate  # Для Windows используйте venv\Scripts\activate

## Установите зависимости:

pip install -r requirements.txt

## Проведите миграции базы данных:

python manage.py migrate

## Запустите сервер разработки:

python manage.py runserver

Теперь вы можете открыть браузер и перейти по адресу http://localhost:8000 🌐
